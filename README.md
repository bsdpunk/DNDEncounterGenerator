# Generate Monster Encounters for DND
## Install and Use
```
#Download And Compile DNDEncounterGenerator
go get github.com/bsdpunk/DNDEncounterGenerator
#Within a folder where dummy.json (Your File with your monsters ) exists, one is in this repo if you don't have your own, run the Generator
DNDEncounterGenerator 500 5
#The 500 represents XP and the 5 represents monster numbers, The DMG only has a table to 15 monsters, so that's all the Program supports
```
